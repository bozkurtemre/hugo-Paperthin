## Features

- Responsive and minimal design
- Syntax highlighting
- Customizable
- SEO optimized
- RSS feed
- Social media links
- Google Analytics
- Table of contents
- Custom CSS
- Custom JS

## Installation

1. Add the theme as a Git submodule:

```bash
git submodule add https://github.com/bozkurtemre/hugo-Paperthin.git themes/Paperthin
```

2. Configure your `hugo.toml`:

```toml
theme = 'Paperthin'
```
